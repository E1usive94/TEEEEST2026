# Новый Вектор 2026 — сайт для GitHub Pages

Это облегчённая версия сайта: оставлены только нужные страницы и ассеты, чтобы GitHub не выдавал ошибку «Yowza, that’s a lot of files».

## Что загружать на GitHub

1. Распакуйте этот архив.
2. Откройте получившуюся папку.
3. Загрузите на GitHub именно содержимое папки, а не сам ZIP.

Файлов меньше 100, поэтому веб-загрузка GitHub должна пройти нормально.

## Главная логика сайта

- `index.html` — главная страница с двумя кнопками: **Моё сегодня** и **Меню**.
- `today.html` — страница-переход на актуальный день.
- `menu.html` — основное меню.
- `program.html` — программа по дням.
- `before.html`, `day1.html` … `day7.html` — страницы дней.
- `materials.html` — материалы и ссылки.
- `team.html` — команда проекта.
- `team-management.html`, `team-districts.html`, `team-centers.html`, `team-services.html` — подразделы команды.
- `map.html` — карта территории.
- `media.html` — медиа.
- `faq.html` — вопросы и ответы.

## Как поменять актуальный день

Откройте файл:

```js
js/config.js
```

И замените:

```js
currentDayUrl: "day1.html"
```

например на:

```js
currentDayUrl: "day4.html"
```

После этого кнопка **Моё сегодня** будет вести на нужный день.
