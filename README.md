# Новый Вектор 2026 — сайт для GitHub Pages

## Структура

Главная страница (`index.html`) содержит только два основных действия:

- **Моё сегодня** — быстрый переход на актуальный день;
- **Меню** — переход к основным разделам сайта.

В меню есть разделы:

1. `program.html` — Программа: До Вектора + День 1–7;
2. `materials.html` — Материалы: ссылки на Яндекс Диск;
3. `team.html` — Команда: управляющая команда, руководители округов, центры, службы;
4. `map.html` — Карта территории;
5. `media.html` — Медиа;
6. `faq.html` — FAQ.

## Как поменять актуальный день для кнопки «Моё сегодня»

Откройте файл:

```js
js/config.js
```

И измените строку:

```js
currentDayUrl: "day1.html"
```

Например:

```js
currentDayUrl: "day4.html"
```

После этого кнопка «Моё сегодня» и страница `today.html` будут вести на нужный день.

## Где менять ссылки на Яндекс Диск и соцсети

- материалы: `materials.html`
- медиа: `media.html`

Сейчас там стоят заглушки `href="#"`. Замените `#` на нужную ссылку.

## Публикация на GitHub Pages

1. Создайте репозиторий.
2. Загрузите все файлы из архива в корень репозитория.
3. Откройте Settings → Pages.
4. Source: Deploy from a branch.
5. Branch: main / root.
6. Сохраните.

Главная страница сайта — `index.html`.
