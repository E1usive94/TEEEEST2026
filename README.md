# Новый Вектор 2026 — сайт

Статический сайт для GitHub Pages.

## Структура
- `index.html` — главный экран: «Моё сегодня» и «Меню»
- `today.html` — автоматический переход на текущий день
- `program.html`, `before.html`, `day1.html`–`day7.html` — программа
- `materials.html` — материалы из `data/materials.json`
- `team.html` — команда
- `map.html` — карта территории
- `media.html` — медиа
- `faq.html` — вопросы из `data/faq.json`

## Как обновлять текущий день
В `js/site.js` и `today.html` задана дата старта `2026-07-02`. При необходимости поменяйте её или вручную измените ссылки.

## Размещение на GitHub Pages
Загрузите содержимое папки в репозиторий, затем включите Pages: Settings → Pages → Deploy from branch → main/root.


## Стиль
Сайт приведён к фирменному тёмному стилю «Новый Вектор 2026»: зелёно-оранжевая палитра, тёмные карточки, акцентная главная страница с бренд-фоном.
